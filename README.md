## File Guide

| File | What it does |
|------|--------------|
| `index.html` | Full single-page portfolio — glassmorphism nav, animated particle canvas, project cards, skills grid, contact section. Zero dependencies; all CSS and JS inline. |
| `myphoto.jpg` | Profile photo used in the hero section |

## Design

- Dark theme with CSS custom properties (`--cyan`, `--purple`, `--mint`, `--pink`)
- Animated canvas particle background via the Canvas API
- Fully responsive with a mobile-first layout
- Smooth scroll navigation with backdrop blur header

## Tech

`HTML5` · `CSS3 (custom properties + grid + flexbox)` · `Vanilla JavaScript` · `Canvas API`
